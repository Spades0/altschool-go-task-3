# altschool-go-task-3
ATM CLI application 
Usage: `go run main.go`

App features 
* Change pin (default pin: 9876) 
* Check account balance (default balance $10000) 
* Deposit funds
* Withdraw funds  
* Exit (selecting this option should exit the program)
